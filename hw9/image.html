1.Image + Width 100vw

+ Привыный метод, хорош для семантики. Альты помогают поисковым запросам

- Если не векторное изображение, то будут потери в качестве

2.background-image

+ Если необходимо поверх написать текст или поместить блоки - удобно использовать фон.

- Фоновое изображение, как полноценное изображение не используется. И нехорошо для семантики

3.Форматы файлов векторных изображений: SVG

+ Не теряется качество,SVG изображениями просто управлять

- Для примитивных рисунков или фигур. Нет поддержки у старых браузеров

4.URL

+ Не теряется качество

- Только для мелкой графики
